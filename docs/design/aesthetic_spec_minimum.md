# ColorPhage 审美规范最小版

## 1. 文档目的

本文件用于定义 ColorPhage 第一阶段开发所需的最小审美规范，为 palette 设计、筛选、命名和后续 API 实现提供统一标准。

本规范服务于以下目标：

- 保证整个包具有一致的审美主张
- 降低 palette 设计过程中的主观摇摆
- 为简单模式、family 模式和后续生成模式提供共同边界
- 让“好看、可区分、符合科研语境”具备可落地的判断依据

## 2. 总体审美定位

ColorPhage 的审美定位不是：

- 纯期刊默认色板复刻
- 纯功能性配色工具
- 商业海报风或互联网 UI 风调色器

ColorPhage 的审美定位是：

- 面向科研论文配图的高完成度配色系统
- 强调高级感、秩序感、区分度和科研适配性
- 让默认输出就具有“像认真设计过”的视觉品质

## 3. 核心判断标准

第一阶段 palette 的筛选和设计应优先满足以下 4 个标准：

### 3.1 好看

- 整组颜色放在一起时应具有整体美感
- 单个颜色不能抢夺整组秩序
- 配色应耐看，而不是只在第一眼显得鲜艳

### 3.2 可区分

- 用于科研图时，不同组别应有足够分离度
- 不能出现多个颜色在常见图形尺寸下难以区分的情况
- 区分度必须建立在整体美感之内，而不是靠粗暴拉大冲突

### 3.3 有科研感

- 图放入论文、补充材料、汇报页中不应显得突兀
- 颜色应服务于信息表达，而不是喧宾夺主
- 整体气质应偏理性、克制、整洁、可信赖

### 3.4 可发表

- 配色在常见论文图尺寸下仍应保持辨识度
- 应适合柱状图、折线图、散点图、热图等常见科研图型
- 应避免明显不适合正式科研语境的视觉效果

## 4. 总体设计原则

### 4.1 追求组合感，不追求颜色堆砌

- 不以 palette 数量多为目标
- 更重视每一组颜色作为整体是否成立
- 色板应体现秩序、节奏和层次

### 4.2 低到中等饱和度优先

- 默认优先低到中等饱和度
- 可以有强调色，但不允许整组长期处于高纯度高刺激状态
- 高饱和色只能作为局部视觉锚点，而不是全员高亮

### 4.3 保留视觉锚点

- 柔和不等于发灰、发脏、无重点
- 即使是 `soft` 或 `mist` 类 family，也应保留至少一个相对清晰的颜色重心

### 4.4 颜色服务于结构

- 颜色首先用于区分组别和引导视线
- 颜色不能压过图形结构本身
- 色板应有助于建立主次，而不是制造噪声

### 4.5 科研感优先于潮流感

- 不追求短期流行的“社交媒体式惊艳”
- 不刻意模仿商业品牌视觉
- 允许有个性，但个性必须在科研语境中成立

## 5. 明确避免的方向

以下方向在第一阶段应默认避免：

- 彩虹色连续渐变
- 多个高纯度亮色同时竞争视觉焦点
- 明显的廉价撞色
- 过强的赛博 UI 风
- 过于品牌化或广告化的视觉印象
- 复刻期刊色板到缺乏个性
- 看起来“很设计”，但放进论文会跳戏的颜色组合

## 6. 简单模式审美规则

### 6.1 `soft`

定位：

- 柔和、精致、克制、耐看

规则：

- 以低到中低饱和度为主
- 明度以中高为主
- 对比控制在低到中等范围
- 冷暖可混合，但整体语气必须统一
- 不允许刺眼亮色成为主导

适用感受：

- 安静
- 精致
- 适合长期观看

### 6.2 `vivid`

定位：

- 鲜明、聚焦、有冲击力，但不俗艳

规则：

- 可使用更大的色相跨度
- 允许中高饱和度颜色出现
- 必须有明确视觉锚点
- 对比强度高于 `soft`
- 不允许发展成“彩色糖豆感”或商业海报感

适用感受：

- 鲜明
- 有表现力
- 适合突出主结论

## 7. family 级审美边界

### 7.1 `mist`

- 低到中低饱和度
- 中高明度
- 柔和、轻盈、安静
- 适合温和分组和主文统计图

### 7.2 `mineral`

- 低到中等饱和度
- 中等明度
- 偏冷、理性、硬质
- 适合临床统计、方法比较、机制图

### 7.3 `forest`

- 中低到中等饱和度
- 中等明度
- 自然、有机、稳健
- 适合生物、生态、微生物相关图

### 7.4 `ocean`

- 中低到中等饱和度
- 中高明度
- 清洁、流动、偏冷
- 适合热图、梯度图、连续变量图

### 7.5 `cinema`

- 中等到中高饱和度
- 明度层次可以拉开
- 戏剧化、聚焦、有叙事感
- 适合突出主结果和组间冲突

### 7.6 `retro`

- 中低到中等饱和度
- 中等偏暖明度
- 温暖、沉静、有时间感
- 适合综述型或叙事感更强的科研图

## 8. 第一阶段 palette 设计要求

第一阶段 palette 设计建议满足：

- 每个 palette 优先覆盖 `n = 3` 到 `n = 8`
- 离散 palette 优先于连续和发散 palette
- 每组颜色在白底科研图中应表现稳定
- 优先考虑常见图型中的可读性，而不是仅看单独色条展示效果

## 9. 第一阶段评估清单

后续每设计一组 palette，可使用以下清单快速判断：

- 放在一起是否整体协调
- 是否至少有一个明确视觉锚点
- 相邻颜色是否足够区分
- 是否像科研图，而不是品牌图或海报图
- 是否和所属 family 的气质一致
- 放进柱状图、折线图、散点图中是否依然成立
